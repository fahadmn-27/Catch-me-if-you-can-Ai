# Catch Me If You Can - AI Maze Game

A dynamic 2D grid game where an AI enemy chases the player using pathfinding algorithms.

## Features
- **Smart AI:** The enemy uses the **BFS (Breadth-First Search)** algorithm to find the shortest path to the player.
- **Dynamic Obstacles:** Walls change positions every 5 seconds, forcing the AI to recalculate its path in real-time.
- **Responsive Gameplay:** Built with Vanilla JavaScript and HTML5.

## How to Play
1. Clone the repository.
2. Open `index.html` in any web browser.
3. Use Arrow Keys to escape from the enemy!
